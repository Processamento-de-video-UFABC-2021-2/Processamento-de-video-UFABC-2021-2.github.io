Detecção de uso de máscara facial

Contexto

Em março de 2020 fomos acometidos pela pandemia originada pelo Sars-CoV-2, o Covid-19. Devido a isso, a maneira com que estávamos costumados a viver mudou de maneira drástica, agora com isolamentos e uso de materias para medida preventiva contra o vírus, sendo o uso da máscara facial que cubra as vias aereas o mais marcante.

Mesmo diante de amplas campanhas mundiais sobre a necessidade do uso de máscara, podemos ver que não é toda população que segue as medidas preventivas, o que contribui para o alongamento da pandemia e a disseminação do vírus.

Cenário do trabalho

Dessa forma, propomos criar uma aplicação que, sob as condições ideias, consiga identificar a presença do uso de máscara tanto em vídeos, visando ser utilizada em locais de grande movimentação, como em catracas de entrada para transporte público, entradas de shoppings e comércios.

> Obtenção de dados

Para confecção da aplicação é necessário criar o corpus que será analisado pelos algorítmos de inteligência artificial.

No primeiro momento, pesquisamos por datasets existentes sobre o tema, porém, pelo fato de se tratar de um cenário jamais visto, houveram dificuldades para encontrarmos bons dados, muitas vezes com imagens de baixa qualidade ou alteradas de maneira digital.

Assim, foi utilizamos a ferramenta Unsplash, auto-denominada como "The internet’s source of freely-usable images", ou seja, uma fonte de imagens para ser usada gratuitamente, livres de quaisquer tipo de direitos autorais. Desse modo, utilizamos sua API, a qual a partir de um parâmetro de busca, retorna os endereços das imagens em diversas resoluções. Para o projeto, o parâmetro de busca utilizado foi "face mask", para obter fotos de pessoas utilizando máscaras faciais. Por conseguinte, foi criado um script na linguagem JavaScript para realizar o downlaod das imagens no tamanho "small", a partir de suas URLs obtidas via API.

Código do script em JS para download das imagens

Após o download das imagens, foi necessário realizar uma revisão sobre as mesmas, pois o parâmetro de busca traz imagens incongruentes ao projeto e, por se tratar de uma automação, não houve um "filtro" inicial. No fim, foi possível construir um corpus com mais de 1.200 imagens.

> Treinamento do classificador via inteligência Artificial

Para criarmos o classificador que identifica o uso de máscara utilizaremos metodologias de "deep learning" via TensorFlow, um framework open source para machine learning.
